# Tech Essay Skill

专业写技术文章的 Claude Code Skill。

## 功能特性

- 🎯 **智能识别文章类型**：自动识别教程、案例、经验、分析四种文章类型
- 📝 **结构化写作指导**：提供专业的文章结构和写作流程
- 🎨 **混合风格支持**：支持专业严谨、代码驱动、故事叙述等多种风格
- ✅ **质量检查清单**：确保文章质量达标

## 项目结构

```
.claude/
├── settings.local.json          # 项目配置
└── skills/
    └── tech-essay/
        ├── SKILL.md             # 主技能文件
        ├── references/
        │   ├── writing-guide.md # 写作技巧指南
        │   └── style-guide.md   # 风格规范
        └── templates/
            ├── tutorial.md      # 教程模板
            ├── case-study.md    # 实战案例模板
            ├── experience.md    # 项目经验模板
            └── analysis.md      # 技术分析模板
```

## 使用方法

在 Claude Code 中，当你说：

- "帮我写一篇关于 React Hooks 的教程"
- "我想分享最近做的性能优化实战"
- "写一篇项目复盘文章"
- "对比一下 Next.js 和 Nuxt.js"

技能会自动触发，提供专业的写作指导。

## 支持的文章类型

| 类型 | 说明 | 关键词 |
|------|------|--------|
| 技术教程 | 帮助读者学会某项技术 | 教程、指南、入门、详解、原理 |
| 实战案例 | 展示具体问题的解决方案 | 实战、解决、修复、实现、踩坑 |
| 项目经验 | 分享项目中的经验教训 | 复盘、总结、经验、心得、体会 |
| 技术分析 | 深入分析技术方案 | 分析、调研、对比、趋势、研究 |
